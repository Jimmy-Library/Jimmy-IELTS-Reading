文章正文

支持 HTML 片段。每段建议用 <p><strong>A</strong> 段落正文…</p> 形式。
如果选用 标题匹配(heading→paragraph) 题型,需要在每段外面包一层 <div class="paragraph-wrapper" id="qN-anchor">,见模板按钮。

题组管理

每个题组对应一种题型(kind)。题号自动按从上到下编号 → q1, q2, q3 …
如需调整顺序,点击题组左侧的"↑ ↓ × 折叠"操作。

    setTimeout(() => el.className = 'toast', 2500);
